## 🚀 Featured Projects

### 🔍 [Cloud Anomaly Detection & Real-Time Infrastructure Monitoring](https://github.com/SameerSyele/cloud-anomaly-detection)
> **ML-powered anomaly detection pipeline for cloud infrastructure metrics with real-time REST API exposure** &nbsp;·&nbsp; `Feb 2026`

Engineered a real-time anomaly detection pipeline using Isolation Forest on cloud infrastructure metrics (CPU, memory, disk, network). Tuned `n_estimators` and contamination factor to optimize precision-recall, and exposed predictions via a FastAPI REST API with automated model retraining on data drift. Built a Streamlit monitoring dashboard with live metric visualization, model confidence scoring, and AI stability indicators — enabling zero-manual-log infrastructure health diagnosis. Implemented version-controlled model persistence in SQLite for continuous learning across data ingestion cycles without prediction downtime.

**Stack:** `Python` `Isolation Forest` `FastAPI` `Streamlit` `SQLite`

### 📈 [AI-Driven Enterprise Cost Optimization & Forecasting System](https://github.com/SameerSyele/cost-forecasting-system)
> **Time-series forecasting engine for enterprise operational cost projection with Flask REST API reporting** &nbsp;·&nbsp; `Nov 2025`

Built a time-series forecasting engine using regression models and grid-search hyperparameter tuning on 10+ financial datasets to project enterprise operational costs and surface budget inefficiencies. Designed a Flask REST API exposing forecast endpoints for downstream dashboard integration, delivering a visual reporting module comparing projected vs. actual costs across business functions. Applied full ETL preprocessing — missing value handling, outlier removal, lag feature engineering — to ensure raw financial data was model-ready and reliable.

**Stack:** `Python` `Scikit-Learn` `Pandas` `Time-Series` `Flask`

### 🤖 [AI Resume Analyzer & Job Skill Gap Detection System](https://github.com/SameerSyele/ai-resume-analyzer)
> **Hybrid NLP + OpenAI engine for resume scoring, skill extraction, and automated candidate screening** &nbsp;·&nbsp; `Aug 2025`

Designed a hybrid NLP matching engine combining a rule-based system with a Scikit-learn classifier and an OpenAI API semantic layer to score resumes, detect skill gaps, and assess job-role compatibility across multiple JDs simultaneously. Built a TF-IDF text preprocessing pipeline (tokenization, stopword removal, entity extraction) and a Flask REST API backend with SQLite persistence for batch resume processing. Delivered an interactive dashboard for resume scoring and skill visualization, automating candidate screening workflows end-to-end.
